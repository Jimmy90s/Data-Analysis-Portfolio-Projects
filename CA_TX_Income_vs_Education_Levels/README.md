
# CA & TX Income vs Education Levels

### Skills Used

- Python
- SQL
- Tableau

### Data Question & Collection

- Compare California and Texas income and education level percentages of 25 year olds or older over the last 5 years.
- US census data was collected from https://data.census.gov by searching S1501 for education levels and DP03 for population data.
- Multiple tables then uploaded to SQL Server with the import wizard.
- Look at meta data to get names for column codes
